# 黃燈灰董事長主題樂園｜品牌資產清單 V1

本清單記錄目前已落地的品牌定型資產。所有 SVG 皆為網站可直接引用的本地檔案。

## Logo / 標誌

- `assets/favicon.svg`
  - 用途：網站 favicon、小尺寸識別。
  - 內容：簡化咖灰頭像、黃燈底色、董事長線條。

- `assets/brand-logo-mark.svg`
  - 用途：圓形園區章、社群頭像、貼紙中心圖。
  - 內容：咖灰圓形徽章與董事長文字。

- `assets/brand-logo-horizontal.svg`
  - 用途：網站 header、品牌頁、對外視覺標題。
  - 內容：圓形園區章、品牌名稱、副標與核心標語。

## 董事長印章系統

- `assets/stamp-read.svg`
  - 狀態：本鳥已閱
  - 用途：公告、事件檔案、已讀批示。

- `assets/stamp-approved.svg`
  - 狀態：暫准通過
  - 用途：互動工具正向結果、商品上架前暫准狀態。

- `assets/stamp-pending.svg`
  - 狀態：待補件
  - 用途：栗米預算、文件不足、需要補充說明。

- `assets/stamp-yellow-alert.svg`
  - 狀態：黃燈警戒
  - 用途：警戒公告、可疑人類行為、蜘蛛鳥預備狀態。

- `assets/stamp-bbb.svg`
  - 狀態：BBB / 不通過
  - 用途：一般不通過章。BBB 是正式稽核語言，不是隨便梗圖。

- `assets/stamp-bbbbbbb.svg`
  - 狀態：BBBBBBB / 嚴重不通過
  - 用途：重大服務缺失、抓癢事故、董事長強烈不滿。

- `assets/stamp-human-improve.svg`
  - 狀態：人類需改善
  - 用途：人類反省區、發言審核、行為評鑑。

## 使用規範

- 印章不可任意改字。
- BBB 與 BBBBBBB 必須維持正式稽核語氣。
- Logo 可縮放，但不要改成高彩度、卡通或普通寵物品牌風。
- 咖灰不可被改成吉祥物、蛋形鳥、麻糬鳥或穿西裝的人類主管鳥。
