# 咖灰董事長正式首頁主視覺規格 V1

用途：作為後續重新生成首頁董事長形象照、調整 Hero 右側主視覺、製作社群圖與延伸視覺時的唯一標準。

參考圖：`../assets/brand-board-v1.png`

## 一、目標

產出一張可以放在首頁 Hero 的正式主視覺，讓使用者第一眼理解：

咖灰不是吉祥物，而是海邊主題樂園的董事長與最高稽核者。

主視覺要同時具備：
- 真實鸚鵡可信度
- 莫蘭迪海邊度假感
- 董事長辦公室情境
- 一本正經的荒謬幽默
- 親切但有審核壓迫感

## 二、圖片尺寸與輸出

建議輸出三種比例：

1. 首頁主卡標準版：1600 x 1600 px，正方形。
2. 桌機 Hero 寬版：2400 x 1600 px，3:2。
3. 手機裁切版：1200 x 1600 px，3:4。

主體安全區：
- 咖灰全身必須在中央 70% 寬度內。
- 頭頂、腳、翅膀邊緣不得被裁切。
- 桌面道具可以靠近邊緣，但不可搶過咖灰。
- 若要只先產一張，優先產 1600 x 1600 px。

檔案建議：
- 原始輸出：PNG。
- 網站使用：WebP 或壓縮 PNG。
- 檔名：`kahui-chairman-hero-v1.png` 或 `kahui-chairman-hero-v1.webp`。

## 三、咖灰姿態

構圖要求：
- 正面。
- 穩重。
- 對稱感。
- 眼神看向鏡頭。
- 微微昂首，帶一點董事長自信。
- 身體完整，不切頭、不切腳。
- 有清楚胸口、翅膀與站姿。

表情要求：
- 親切但有審核壓迫感。
- 不要兇惡。
- 不要賣萌過頭。
- 不要歪頭媚態。
- 不要像證件照，要像董事長在海邊辦公室坐鎮。

比例要求：
- 保留真實鸚鵡比例。
- 圓潤但不能變成蛋、麻糬、玩偶。
- 頭部和身體可圓潤，但輪廓要有真鳥結構。

## 四、咖灰外觀

必須保留：
- 頭部偏奶油灰白。
- 身體低飽和灰藍。
- 翅膀略帶灰綠或深灰藍。
- 黑色圓眼。
- 粉膚色鳥喙。
- 羽毛細節柔順。
- 站姿穩定，有腳爪。

禁止：
- 髮髻。
- 眉毛。
- 人類五官。
- 西裝。
- 領帶。
- 過度擬人化服裝。
- 腹部名牌成為視覺中心。
- 變成胖蛋形角色。
- 變成卡通 mascot。

## 五、背景情境

場景：海邊主題樂園中的董事長辦公室。

必要元素：
- 乾淨海平線。
- 柔和海風感。
- 海鹽白與莫蘭迪灰藍背景。
- 黃燈暖光。
- 小型辦公桌。
- 栗米報表。
- 小印章。
- 紅土補給罐。

可選元素：
- 木質園區招牌。
- 園區窗景。
- 植栽。
- 文件夾。
- 小鳥腳印。
- 低調 BBB 印章。

背景限制：
- 背景乾淨，不要過度堆滿道具。
- 黃燈可以明顯，但不要壓過咖灰。
- BBB 是配件，不是畫面主角。
- 不要讓桌上物件遮住咖灰腳部。

## 六、構圖規格

標準正方形構圖：
- 咖灰置中，佔畫面高度約 58% 到 68%。
- 咖灰眼睛位於畫面高度約 34% 到 42%。
- 桌面位於畫面下方 20% 到 28%。
- 海平線位於畫面高度約 36% 到 45%。
- 黃燈放在左側或右側，不要貼邊。
- 招牌可在上方，但文字不可比咖灰更搶眼。

首頁使用時的安全裁切：
- 桌機右側卡：以正方形完整呈現。
- 手機版：優先保留咖灰頭、身體、腳與桌面核心道具。
- 不要使用只剩臉部的大頭裁切。

## 七、文字與標誌

可以出現的文字：
- 黃燈灰董事長主題樂園
- 黃燈警戒中
- 栗米報表
- 小印章
- 紅土補給罐
- 本鳥已閱
- BBB

避免：
- 大量小字。
- 文字糊成不可讀的招牌。
- 把咖灰掛胸牌當主視覺焦點。
- 讓 BBB 比咖灰更醒目。

## 八、印章系統

主視覺可用的印章：
- 本鳥已閱
- 暫准通過
- 待補件
- 黃燈警戒
- BBB

不建議在首頁主視覺放：
- 太大的 BBBBBBB。
- 多個大型印章重疊。
- 紅色過強的警告章。

首頁主視覺的印章語氣應該是「正式辦公室配件」，不是梗圖主角。

## 十一、驗收標準

合格主視覺應符合：
- 第一眼看得出咖灰是主角。
- 咖灰不像每次生成不同角色。
- 有真實鸚鵡比例與清楚站姿。
- 有海邊董事長辦公室情境。
- 莫蘭迪色系明確。
- 黃燈與辦公桌道具支撐世界觀。
- BBB 是配件，不是主角。
- 適合放在首頁 Hero 右側。
- 手機裁切後仍能保留咖灰全身重點。

不合格狀況：
- 咖灰變成蛋或麻糬。
- 咖灰看起來像吉祥物。
- 咖灰穿人類主管衣服。
- 鏡頭只剩臉部。
- 背景和道具比咖灰更搶眼。
- 缺少海邊或董事長辦公室感。
- 文字太多導致畫面像海報而不是主視覺。
